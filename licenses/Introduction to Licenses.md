---
Subject: Licenses
---
> [!INFO] 
> This note will be talking in depth about mostly the open source licenses that most FOSS projects use on GitHub.

There are two types of these licenses, permissive and copyleft:

**Permissive**: Permissive licenses are designed to provide maximum freedom to use, distribute, or modify software with minimal restrictions, you do not have to make your project open source to use the code. Common licenses include the [[MIT License]], [[Apache License 2.0]], and the BSD License.

**Copyleft**: Copyleft licenses are designed that you can use, modify, or distribute the code, but you must make the project open source, or give the source code to the original author. The most popular example of this is the [[GPL License v3.0]]
