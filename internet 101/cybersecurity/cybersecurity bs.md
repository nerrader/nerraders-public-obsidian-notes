Security starts in the [[BIOS UEFI|BIOS/UEFI]], not on windows. If your BIOS is open, then your computer is just a toy.

So the first step is to set a password for each of them because [[Cold Boot Attacks|cold boot attacks]] are real. As if you have a [[BIOS UEFI|BIOS/UEFI]] Password, it locks the boot order, which kills the hijack before it even starts.

Enable secure boot (if youre on windows), as these are your first line of defense against [[Types of Computer Viruses#**Rootkits**|Bootkits]].
